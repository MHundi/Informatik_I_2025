Überprüfen Sie, ob Python die Rechenregel ``Punkt-vor-Strich'' erfüllt.

\DecValTok{3}\OperatorTok{+}\DecValTok{4}\OperatorTok{*}\DecValTok{5}

korrekt aus.

\subsubsection{Aufgabe 2}\label{aufgabe-2}

In Python gibt es neben den Grundrechenarten +, -, *, / für \texttt{int}
oder \texttt{float}-Zahlenwerte auch weitere Rechenoperatoren.
Untersuchen Sie diese wie angegeben:

\begin{enumerate}
\def\labelenumi{\alph{enumi})}
\item
  \texttt{a//b}
\item
  \texttt{a\%b}
\item
  \texttt{a**b}
\end{enumerate}

Vorgehensweise:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Wählen Sie passende Werte für a und b (int oder float) und berechnen
  Sie jeweils die Ergebnisse in Python.
\item
  Erklären Sie, was der Operator bewirkt.
\item
  Geben Sie eine passende Bezeichnung für jeden Operator an.
\item
  Überlegen Sie ein Beispiel aus dem Alltag, wo der Operator sinnvoll
  wäre.
\end{enumerate}

\begin{Shaded}
\begin{Highlighting}[numbers=left,,]
\CommentTok{\#Aufgabe a)}
\DecValTok{17} \OperatorTok{//} \DecValTok{5}   \CommentTok{\# ergibt 3}
\end{Highlighting}
\end{Shaded}

\begin{verbatim}
3
\end{verbatim}

\begin{itemize}
\tightlist
\item
  \textbf{Erklärung}: Ganzzahlige Division -- das Ergebnis wird
  abgerundet auf die nächste ganze Zahl.
\item
  \textbf{Bezeichnung}: \emph{Ganzzahlige Division}
\item
  \textbf{Alltagsbeispiel}: Wie viele volle Päckchen mit 5 Keksen kann
  man aus 17 Keksen machen?
\end{itemize}
